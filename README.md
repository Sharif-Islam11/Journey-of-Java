# Java Basics Project

## About This Project
This project explains the basics of Java programming and made for beginners to understand the basics.

## Topics Covered

### 1. Print Output
Output means showing data to the user.
We use System.out.println().
Example:
System.out.println("Hello World");

### 2. Taking input from users
Input means taking data from the user.
We use the Scanner class in Java.
Example:
import java.util.Scanner;

Scanner input = new Scanner(System.in);
int number = input.nextInt();
